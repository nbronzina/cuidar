# Mapa de terminología — Cuidados en Red

Tabla completa de términos correctos e incorrectos para el sitio.

## Personas y roles

| Correcto | Incorrecto | Notas |
|---|---|---|
| persona mayor | anciano/a, viejo/a, abuelo/a, tercera edad, adulto en plenitud | "Adulto mayor" solo en docs técnicos |
| persona cuidadora | cuidador/a (en sitio público) | "Cuidadora" aceptable en docs técnicos |
| persona beneficiaria | paciente, cliente | "Usuario/a" aceptable en contexto informal |
| familiar, persona de referencia | pariente, deudo | — |
| equipo interdisciplinario | staff, equipo médico | Incluye trabajadores sociales, no solo médicos |
| trabajador/a social | asistente social | Término profesional actual |

| Correcto | Incorrecto | Notas |
